PTV-Parser

Java library and apps to process EOD market data

View on GitHub Download .zip Download .tar.gz

This program is new and in state "Work in Progress".

The apps I have created work but the software is in no way "stable".

I used data from www.eoddata.com that is free.

The application AppTemplate in net.ajaskey.market.ta.apps is a good place to learn how to use the data input contained within this project. The other apps provide additional ways of processing the input data.

Examples of some of the outputs generated can be found here: https://andysmarketmusings.net
