PTV-Parser

Library and apps to process EOD market data

View on GitHub Download .zip Download .tar.gz

This program is new and in state "Work in Progress".

I used data from www.eoddata.com that is free.

I have used TA-Lib (ta-lib.org) but the plan is that any library can be used if it meets the processing interface specification. I have also wrote some of my own metric calculations.

If you have questions or are interested in working on this project then send an email to askeyaj@gmail.com.
